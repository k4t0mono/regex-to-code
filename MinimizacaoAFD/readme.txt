A linguagem escolhida pelo grupo foi Python.

Para executar o programa deve-se utilizar a seguinte linha de comando (estando no mesmo diretório os arquivos Minimiza.py, Estruturas.py e LeituraEscrita.py):

python Minimiza.py <arquivoEntrada> <arquivoSaida1> <arquivoSaida2>

Onde o <arquivoEntrada> é o arquivo contendo o autômato a ser minimizado, <arquivoSaida1> é o arquivo onde deve-se armazenar a tabela de minimização e o <arquivoSaida2> é onde deve-se armazenar o autômato minimizado.
